# ShiftMelt 🎬

O **ShiftMelt** é uma aplicação desktop desenvolvida em Python focada na automação de download e conversão de média. O projeto oferece uma interface simples para descarregar vídeos do YouTube e converter ficheiros entre diferentes formatos de áudio e vídeo localmente.

## 🚀 Funcionalidades
* **Download de Vídeos:** Descarrega vídeos do YouTube via URL em alta qualidade.
* **Conversor de Média:** Extração de áudio (MP3) de vídeos ou conversão entre formatos populares.
* **Automação:** Processamento ágil de ficheiros sem necessidade de ferramentas online.

## 📥 Como Descarregar e Usar

Existem duas formas de utilizar o ShiftMelt:

### 1. Usuário Comum (Executável)
Se pretendes apenas utilizar a ferramenta sem instalar o Python:
1. Vai à secção [Releases](https://github.com/LuizZaim/ShiftMelt/releases) no lado direito desta página.
2. Descarrega o ficheiro `SHIFTMELT.exe`.
3. Executa o ficheiro diretamente no Windows.

### 2. Programador (Código-fonte)
Se pretendes explorar o código ou rodar via terminal:
1. Clona este repositório: `git clone https://github.com/LuizZaim/ShiftMelt.git`
2. Instala as dependências: `pip install -r requirements.txt`
3. Executa o programa: `python main.py`
